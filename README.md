# Python Quiz Application

## Overview
A simple, interactive console-based quiz game built in Python. The application presents users with multiple-choice questions, processes their input, provides immediate feedback, and displays a final score at the end.

## Features
* **Multiple Choice Questions**: Presents users with a question and 4 possible options.
* **Input Validation**: Accepts user input and compares it to predefined correct answers.
* **Score Tracking**: Keeps track of the total number of correct answers.
* **Immediate Feedback**: Tells the user if they were right or wrong after each question.

## Technologies Used
* Python 3
* Core programming concepts (Loops, Conditionals, Lists, Dictionaries)
